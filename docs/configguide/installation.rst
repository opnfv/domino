Domino Installation
===================

Prerequisites
-------------
git
python-pip
python (version =2.7)
tosca-parser (version >=0.4.0)
heat-translator (version >=0.5.0)

Installation Steps
------------------

Step-0: Prepare Environment

.. code-block:: bash

  > $sudo pip install tosca-parser
  > $sudo pip install heat-translator
